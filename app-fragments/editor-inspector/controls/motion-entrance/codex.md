# Codex: Animation Logic

## Mapping
```javascript
const map = {
  'fade-up': 'animate-fade-up',
  'slide': 'animate-slide',
  'scale': 'animate-scale',
  'rotate': 'animate-rotate',
  'reveal': 'animate-reveal',
  'none': ''
};
```

## Preview
The preview element should have `animation-fill-mode: both;`.
