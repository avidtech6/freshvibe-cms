# Codex: Trigger Logic

## States
- `load`: Triggered once on DOMContentLoaded.
- `scroll`: Triggered on scroll (likely debounced in JS).
- `hover`: Triggered on `mouseenter`/`mouseleave`.
- `click`: Triggered on click.

## Labeling
- `load` -> "On Page Load"
- `scroll` -> "On Scroll"
- `hover` -> "On Hover"
- `click` -> "On Click"
